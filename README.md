## ğŸ”§ Installation & Setup

### Prerequisites
- Flutter SDK (3.0+)
- Dart SDK (2.18+)
- Firebase CLI
- Git

### Quick Start
```bash
# Clone the repository
git clone https://github.com/uriel-academy/uriel-website.git
cd uriel-website

# Install dependencies
flutter pub get

# Configure Firebase
firebase login
firebase use --add
flutterfire configure

# Run in development mode
flutter run -d chrome

# Build for production
flutter build web --release
firebase deploy --only hosting
```

### Environment Setup
1. **Firebase Project**: Create a new Firebase project
2. **Authentication**: Enable Email/Password and Google Sign-In
3. **Firestore**: Set up database rules and indexes
4. **Storage**: Configure storage buckets and security rules
5. **Functions**: Deploy Cloud Functions for backend logic
